Sensor

Sensor Management and Get Sensor Data

센서 등록 API

센서 등록 시, 센서 정보 리턴

Request Body schema: application/json
required

센서 등록 할 때 사용하는 스키마

agentId
required
string <uuid>

센서 에이전트 식별자

name
required
string

센서 이름

description
string

센서 설명

sensorType
required
string

센서 타입 정보 (ex. 레이더, 카메라, 초음파)

dataType
required
string
Enum: "ARRAY_INTEGER" "ARRAY_FLOAT" "ARRAY_STRING" "FLOAT" "INTEGER" "STRING"

센서 데이터 타입 정보 (ex. ARRAY_INTEGER - 리스트의 구성요소가 정수, ARRAY_FLOAT - 리스트의 구성요소가 실수, ARRAY_STRING - 리스트의 구성요소가 문자열, FLOAT - 단순 실수형, INTEGER - 단순 정수형, STRING - 단순 문자열)

dataShape
required
Array of integers

센서의 데이터 차원 (단순 데이터일 경우 [1])

batchSize
required
integer

센서 데이터 배치 수집 시, 벌크 갯수 단위

batchTime
required
integer

센서 데이터 배치 수집 시, 벌크 시간 단위 (초)

Responses
